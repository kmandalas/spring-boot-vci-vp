Wallet-Initiated Issuance

Unlike typical EUDI PoCs that rely on credential offers from the issuer, this project follows a wallet-initiated flow. The wallet presents pre-configured credential types, the user selects one, and the Authorization Code flow brings them to this login screen to authenticate before credential issuance.
